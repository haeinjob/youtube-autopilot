## 1. Cialdini 설득의 7원칙

> 출처: Robert Cialdini, *Influence: The Psychology of Persuasion* (1984, 개정판 2021)
> 35년간 동료 검토 과학적 연구 기반

### 원칙 1: 상호성 (Reciprocity)
**핵심**: 먼저 줘라. 받은 사람은 돌려줘야 한다는 의무감을 느낀다.
- 공짜 가치 제공 → 구매 전환율 상승
- **서비스 적용**: 무료 분석 1회 제공 → 유료 전환 유도
- **예시 공식**: "지금 바로 무료로 내 아이템 시장성 확인해보세요" → 결과 본 후 "더 알고 싶다면..."

### 원칙 2: 일관성 & 헌신 (Commitment & Consistency)
**핵심**: 한번 입장을 취하면, 그 입장과 일치하게 행동하려 한다.
- 작은 동의 → 큰 동의로 이어짐
- **서비스 적용**: 고정댓글 "이 영상 도움됐으면 댓글로 알려주세요" → 댓글 → 구매 전환
- **예시 공식**: "지금 아이템 선택했다면, 이미 절반은 시작한 거예요"

### 원칙 3: 사회적 증거 (Social Proof)
**핵심**: 불확실할 때 남들이 하는 대로 따른다. 숫자가 많을수록 강력.
- **서비스 적용**: "이미 OOO명이 이 아이템으로 수익화 시작" 문구
- **예시 공식**: "지난달 이 카테고리에서 37명이 첫 수익을 냈습니다"

### 원칙 4: 권위 (Authority)
**핵심**: 전문가, 자격증, 숫자, 연구 결과를 신뢰한다.
- **서비스 적용**: "수백만 조회수 데이터 분석 기반", 구체적 통계 수치 제시
- **예시 공식**: "월 500만원 이상 수익화한 크리에이터 데이터 분석 결과"

### 원칙 5: 호감 (Liking)
**핵심**: 좋아하는 사람의 말을 따른다. 유사성, 칭찬, 친숙함이 호감을 만든다.
- **서비스 적용**: 친근한 톤, "당신처럼 처음 시작한 사람들이..." 공감 언어
- **예시 공식**: 동네 형·언니 톤으로 말 걸기

### 원칙 6: 희소성 (Scarcity)
**핵심**: 희귀하거나 곧 사라지는 것은 더 가치 있어 보인다.
- **서비스 적용**: "이 카테고리 지금 경쟁자 적음 — 진입 타이밍 지금이 최적"
- **예시 공식**: "이 아이템 트렌드 올라가는 중 — 지금이 골든타임"

### 원칙 7: 단결/통합 (Unity) — 2021년 추가
**핵심**: "우리" 정체성 공유. 같은 집단임을 느끼면 강력한 영향.
- **서비스 적용**: "유튜브로 수익화하려는 우리끼리만 아는 공식"
- **예시 공식**: "1인 크리에이터라면 이 방법만 알면 됩니다"

## 2. 카네만 행동경제학

> 출처: Daniel Kahneman, *Thinking, Fast and Slow* (2011), Amos Tversky 공동 연구

### System 1 vs System 2

| | System 1 (빠른 사고) | System 2 (느린 사고) |
|--|---------------------|---------------------|
| 속도 | 즉각적, 자동 | 느림, 의도적 |
| 특징 | 감정, 직관, 이미지 | 논리, 분석, 숫자 |
| 마케팅 적용 | 썸네일, 첫인상, 감정 | 상세 설명, 가격 비교 |
| **핵심** | **구매의 95%는 System 1** | 사후 합리화에 사용 |

**서비스 적용**: 썸네일·제목은 System 1 자극 (감정, 이미지, 직관)
상세 페이지는 System 2 충족 (데이터, 근거, 논리)

### 손실 회피 (Loss Aversion)
**핵심**: 손실의 고통 = 동일 크기 이익의 기쁨 × **2.5배**
- 잃는 것에 대한 두려움이 얻는 것의 기쁨보다 강력
- **공식**: "얻을 수 있다"보다 "잃지 않으려면" 프레임이 더 강력

**예시 변환**:
- ❌ "이 방법으로 월 300만원 벌 수 있어요"
- ✅ "지금 이 방법 모르면 경쟁자에게 시장 다 뺏깁니다"

### 10대 핵심 인지 편향 (마케팅 적용)

| 편향 | 정의 | 서비스 적용 |
|------|------|------------|
| **앵커링** | 첫 숫자가 기준점이 됨 | "원가 50만원 → 지금 29,900원" |
| **프레이밍** | 같은 내용도 틀에 따라 다르게 인식 | "실패율 95%" vs "성공한 5%의 비밀" |
| **매몰비용** | 이미 투자한 것에 집착 | "이미 유튜브 시작했다면, 이제 수익만 남았다" |
| **가용성 편향** | 쉽게 떠오르는 것을 더 빈번하다 인식 | 구체적 성공 사례 반복 제시 |
| **FOMO** | 놓칠까봐 두려움 | "이 타이밍 놓치면 1년 후 후회" |
| **밴드왜건** | 많은 사람이 하면 따라함 | "지금 이 카테고리 진입하는 크리에이터 폭발적 증가" |
| **디폴트 편향** | 기본값을 그대로 유지하려 함 | 프로 플랜을 기본 선택으로 표시 |
| **선택 과부하** | 선택지가 너무 많으면 선택 안 함 | 옵션 3개 이하로 제한 |
| **체감 가치** | 가격 아닌 '가치 대비 가격' 인식 | 시간 절약 + 돈 절약으로 가치 표현 |
| **현상 유지 편향** | 변화를 꺼림 | "지금 하는 방식 그대로면 1년 후도 똑같다" |

## 3. 조나 버거 STEPPS

> 출처: Jonah Berger, *Contagious: Why Things Catch On* (2013), 와튼 스쿨 마케팅 교수

### STEPPS 6요소

| 요소 | 정의 | 유튜브 제목 적용 |
|------|------|----------------|
| **S** Social Currency | 공유하면 나를 똑똑/특별해 보이게 함 | "아는 사람만 아는", "전문가들이 숨기는" |
| **T** Triggers | 일상 속 자동 연상 자극 | 계절·이슈·트렌드와 연결 |
| **E** Emotion | 강한 감정(분노, 경이, 공포, 웃음) | 고조 감정 유발 — 잔잔한 감정 X |
| **P** Public | 공개적일수록 모방됨 | "내가 이렇게 해서 됐다" 공개 사례 |
| **P** Practical Value | 실용적 가치 = 공유할 이유 | "지금 당장 쓸 수 있는" 정보 |
| **S** Stories | 스토리 속에 메시지 숨기기 | 나의 변화 서사 = 시청자의 미래 |

**핵심 인사이트**: 가장 바이럴한 콘텐츠는 STEPPS 3개 이상을 동시에 충족

**예시 — 러닝화 판매 채널**:
- S: "마라톤 서브4 달성자들만 아는 훈련법" (Social Currency)
- E: "6개월 만에 무릎 망가졌던 내가 완주한 이야기" (Emotion)
- P: "지금 당장 내일 러닝에 적용할 수 있는 페이스 공식" (Practical Value)

## 4. 카피라이팅 공식 전집

### 4-1. AIDA (가장 오래된 공식, 1898년)
```
A — Attention  : 즉각 멈추게 하는 충격/호기심/질문
I — Interest   : "이건 나 얘기다" 공감 연결
D — Desire     : 갖고 싶게 만드는 결과/변화 묘사
A — Action     : 지금 당장 해야 할 한 가지
```

**유튜브 제목 AIDA 적용**:
- A: "월급쟁이가 유튜브로 월 300만원?" (Attention)
- 썸네일이 I+D 역할: 변화된 모습 or 충격 표정
- 영상 마지막 CTA가 Action

### 4-2. PAS (가장 강력한 전환 공식)
```
P — Problem   : 고통을 구체적으로 명명
A — Agitation : 그 고통을 더 생생하게 증폭 (미래의 공포 포함)
S — Solution  : 내 상품/서비스가 해결책
```

**예시**:
- P: "유튜브 6개월째 조회수 100뷰 못 넘는 분들"
- A: "이 상태면 1년 후에도 똑같습니다. 경쟁자들은 지금 이 방법으로 치고 나가고 있는데"
- S: "오늘 영상에서 조회수 터지는 제목 공식 5가지 알려드립니다"

### 4-3. BAB (Before-After-Bridge, 변화 서사)
```
Before : 지금 이 사람들의 현실 (고통/한계)
After  : 내 솔루션 사용 후의 이상적 결과
Bridge : 어떻게 그 결과에 도달하는가
```

**랜딩페이지 적용**:
- Before: "유튜브 1년째 수익 0원. 뭘 팔아야 할지조차 모름"
- After: "3개월 만에 월 200만원 수익화 달성"
- Bridge: "AI가 아이템 선정부터 대본까지 전부 만들어드립니다"

### 4-4. 4U 헤드라인 공식 (Michael Masterson)
```
Urgent      : 지금 당장 행동해야 하는 이유
Unique      : 다른 데서 못 본 것
Useful      : 실제로 도움이 되는 것
Ultra-specific : 모호함 제거, 숫자와 구체성
```

**나쁜 헤드라인**: "유튜브 수익화 방법"
**좋은 헤드라인**: "지금 이 글 읽는 30분 안에 내 채널 수익화 아이템 찾는 법" (Urgent+Unique+Useful+Ultra-specific)

### 4-5. 추가 공식들

**FAB (Feature → Advantage → Benefit)**
- Feature(기능) → "AI 자동 생성"
- Advantage(장점) → "30초 만에 완성"
- Benefit(혜택) → "내 시간 아끼고 더 많은 콘텐츠 올릴 수 있음"

**4P (Picture, Promise, Prove, Push)**
- Picture: 이상적 미래 그리기
- Promise: 우리가 약속하는 결과
- Prove: 증거 (데이터, 후기)
- Push: 지금 행동 촉구

## 5. 유진 슈워츠 고객 인식 5단계

> 출처: Eugene Schwartz, *Breakthrough Advertising* (1966) — 카피라이팅 역사상 가장 중요한 책

### 5단계 모델

```
1단계 [완전 무인식]  → 문제 존재조차 모름
2단계 [문제 인식]    → 문제는 알지만 해결책 모름
3단계 [솔루션 인식]  → 해결책 있다는 건 알지만 내 것을 모름
4단계 [상품 인식]    → 내 상품 알지만 확신 없음
5단계 [최고 인식]    → 살 준비 완료, 한 방만 더 필요
```

### 단계별 콘텐츠 전략

| 단계 | 풀링 콘텐츠 제목 예시 | 키 콘텐츠 제목 예시 |
|------|---------------------|-------------------|
| 1단계 | "직장 다니면서 부업 안 하면 생기는 일" | - |
| 2단계 | "유튜브로 돈 버는 사람들의 공통점" | - |
| 3단계 | "AI 자동화로 유튜브 수익화 하는 방법" | - |
| 4단계 | - | "이 서비스 쓰고 3개월 만에 월 200만원" |
| 5단계 | - | "지금 가입하면 첫 달 무료 — 내일부터 가격 오름" |

**핵심**: 대부분의 유튜브 시청자 = 1~2단계. 풀링 콘텐츠는 1~2단계 타겟, 키 콘텐츠는 3~4단계 타겟

## 6. 알렉스 호르모지 가치 방정식

> 출처: Alex Hormozi, *$100M Offers* (2021)

### 가치 방정식

```
           꿈의 결과 × 달성 가능성
가치 =  ─────────────────────────────
              시간 지연 × 노력/희생
```

**4개 레버**:
1. **꿈의 결과 높이기**: "월 300만원" → "1년 후 퇴사 가능한 수익"
2. **달성 가능성 높이기**: 보증, 성공 사례, 데이터
3. **시간 지연 줄이기**: "30초 만에", "오늘 바로", "즉시"
4. **노력/희생 줄이기**: "버튼 하나로", "복붙만 하면", "생각할 필요 없이"

**그랜드슬램 오퍼 4요소**:
1. 거부할 수 없는 가치 제안
2. 구매력 있는 굶주린 시장
3. 가치를 강화하는 프리미엄 가격
4. 모든 리스크를 제거하는 보증

**서비스 적용**:
- ❌ "AI로 유튜브 제목 만들어드립니다"
- ✅ "버튼 하나로 조회수 터지는 제목 10개 즉시 생성 — 안 되면 환불"

## 8. 신경마케팅

> 출처: Antonio Damasio, Harvard DCE 연구, iMotions 신경마케팅 연구

### 핵심 발견

> **"우리는 생각하는 기계가 아니라, 느끼는 기계다"** — Antonio Damasio

- **95%의 구매 결정은 무의식(감정)에서 일어난다**
- 소마틱 마커 이론: 감정이 이성보다 먼저 결론을 내린다
- 뇌는 논리적 분석 전에 감정적 반응으로 "살까/말까"를 결정

### 마케팅 적용 원칙

1. **감정 먼저, 논리는 나중**: 헤드라인은 감정 자극, 상세 내용은 논리 보강
2. **고통 회피 = 최강 동기**: 쾌락 추구보다 고통 회피 동기가 2.5배 강함
3. **스토리 = 뇌 동기화**: 이야기를 들으면 청자의 뇌가 화자 뇌와 동기화됨
4. **신체 감각 언어**: "무겁다", "뜨겁다", "날카롭다" 등 감각 언어가 감정 뇌 직접 자극
5. **패턴 인식**: 뇌는 에너지 절약을 위해 패턴을 찾음 → 기대를 깨거나 확인해줄 때 집중

### 소비자 결정 3단계 (신경과학 기반)

```
1. 감정 반응 (0.3초): "이거 좋아/싫어"
2. 직관적 판단 (2초): "살까/말까" — System 1
3. 사후 합리화 (30초+): "왜 샀는지" 이유 만들기 — System 2
```

**서비스 적용**: 첫 3초(훅)가 감정 반응 결정 → 훅이 전부

## 9. 데이비드 오길비 원칙

> 출처: David Ogilvy, *Ogilvy on Advertising* (1983), *Confessions of an Advertising Man* (1963)
> "광고의 아버지" — 실제 광고 집행 결과 데이터 기반

### 핵심 38원칙 중 서비스 적용 상위 10개

1. **헤드라인에 80센트를 써라**: 카피 가치의 80%는 헤드라인. 제목이 전부.
2. **독자의 지능을 존중하라**: 우둔하게 설명하면 신뢰 잃음
3. **큰 약속을 하라**: "Large promise is the soul of an advertisement"
4. **구체성이 설득력**: 모호한 주장보다 숫자와 구체적 사실
5. **스토리텔링**: 이야기가 있는 이미지가 없는 이미지보다 압도적으로 주목받음
6. **숫자는 숫자로**: "세 가지" 말고 "3가지"
7. **짧은 문장, 짧은 단락**: 읽기 쉬울수록 설득력 높아짐
8. **혜택을 팔아라, 기능 말고**: "카메라"가 아닌 "소중한 순간을 영원히"
9. **테스트를 멈추지 마라**: 감에 의존하지 말고 데이터로 검증
10. **첫 문장이 두 번째를 읽게 해야**: 모든 문장의 역할은 다음 문장 읽기

### 오길비 헤드라인 공식 (실제 사용한 패턴)

```
[숫자] + [구체적 결과/방법] + [독자 관련성]
예: "어떻게 나는 광고 한 편으로 197,000달러를 벌었나"

[HOW TO] 공식
예: "어떻게 비전문가도 30일 만에 수익화를 시작할 수 있나"

[공포 + 해결] 공식
예: "당신의 유튜브가 실패하는 진짜 이유 — 그리고 해결법"
```

## 11. 스토리브랜드 (StoryBrand)

> 출처: Donald Miller, *Building a StoryBrand* (2017, 2.0판 2024)
> Joseph Campbell의 영웅의 여정 + 현대 마케팅 결합

### 핵심 패러다임 전환

```
❌ 기존: 브랜드가 주인공 — "우리 회사는 최고입니다"
✅ StoryBrand: 고객이 주인공, 브랜드는 안내자 — "당신의 문제를 해결해드립니다"
```

### 7부작 구조 (SB7 Framework)

```
1. 주인공 (A Character)   : 고객이 원하는 것
2. 문제 (Has a Problem)   : 외부 문제 + 내부 감정 + 철학적 문제
3. 안내자 만남 (Meets a Guide) : 공감 + 권위
4. 계획 제시 (Who Gives a Plan) : 단순 명확한 3단계
5. 행동 촉구 (And Calls Them to Action) : CTA
6. 실패 회피 (That Helps Them Avoid Failure) : 하지 않으면 어떻게 되나
7. 성공 (And Ends in a Success) : 구체적 변화된 삶
```

### 3층 문제 구조 (가장 강력한 카피 공식)

```
외부 문제 (External): "유튜브 조회수가 안 나온다"
내부 감정 (Internal): "나는 콘텐츠 재능이 없는 건가" (자존감)
철학적 문제 (Philosophical): "열심히 하는 사람이 보상받아야 한다"
```

**서비스 카피 적용**:
- 외부: "뭘 팔지 모르겠다"
- 내부: "나만 못하는 건가, 나는 사업 감각이 없는 건가"
- 철학: "노력하는 사람은 기회를 잡을 수 있어야 한다"

## 12. 훅 공식 — 3초의 법칙

> 출처: Facebook Research, Buffer, OpusClip 2025 데이터

### 핵심 데이터

- 첫 **1.3초** 안에 잡지 못하면 스크롤
- 첫 3초 시청 → 10초 시청 전환율: **65%**
- 첫 3초 시청 → 30초 시청 전환율: **45%**
- 호기심 갭 훅 → 완료율 +**234%**, 공유율 +**178%**
- 무음 시청 비율: **85%** → 자막 필수

### 훅 유형 5가지

**1. 패턴 인터럽트 (Pattern Interrupt)**
- 예상을 깨는 시작
- "저 오늘 사직서 냈습니다" (직장인 채널에서)
- "3개월 만에 팔로워 0명에서 10만 됐는데, 사실 이렇게 한 거 아닙니다"

**2. 호기심 갭 (Curiosity Gap)**
- 정보의 일부만 공개, 나머지는 영상 안에
- "이 한 가지만 바꿨더니 조회수가 10배 됐습니다" (뭔지 안 말함)
- "제가 6개월 동안 숨겼던 수익화 비밀" (뭔지 공개 안 함)

**3. 증거 먼저 (Proof First)**
- 결과를 먼저 보여주고 "어떻게?"를 유발
- 통장 캡처, 조회수 화면, 전후 비교
- "이게 제 지난달 수익입니다" (숫자 먼저 공개)

**4. 충격 선언 (Shocking Statement)**
- 통념과 반대되는 주장
- "구독자 많아도 돈 못 버는 이유" (구독자 = 수익이 아님을 암시)
- "조회수 1000뷰인데 월 200만원 버는 방법"

**5. 공감 + 고통 (Relatable Pain)**
- 시청자의 고통을 정확히 말함
- "유튜브 6개월 했는데 아직 구독자 100명인 분들"
- "매일 올려도 조회수 세 자리 못 넘는 답답함"

### 훅 작성 공식

```
[시청자 고통 명시] + [반전/충격] + [약속]

예: "유튜브 1년 했는데 수익 0원인 분들 (고통)
    — 오늘 이 방법 알면 (반전 예고)
    다음 달부터 달라집니다 (약속)"
```

## 13. 서비스 적용 매핑표

> 서비스 출력물별로 어떤 프레임워크를 자동 적용할지 정의

### 유튜브 제목 생성 시 적용 우선순위

```
1순위: 호기심 갭 (Curiosity Gap) — 완료율 +234%
2순위: 손실 회피 (Loss Aversion) — 이익 대비 2.5배 효과
3순위: 사회적 증거 (Social Proof) — 숫자, 성공 사례
4순위: 패턴 인터럽트 — 통념 깨기
5순위: 4U 공식 — Ultra-specific 적용
```

### 유튜브 대본 구조 적용

```
훅 (0-15초):   패턴 인터럽트 or 공감+고통 → System 1 자극
문제 (15-60초): PAS의 P+A — 고통 명명 + 증폭
해결 (60초+):  BAB의 Bridge — 솔루션 제시
증거 (중반):   사회적 증거 + 권위 (Cialdini 3+4)
클로징:        희소성 + CTA (Cialdini 6 + AIDA의 A)
```

### 랜딩페이지 카피 적용

```
헤드라인:     4U + 오길비 "큰 약속" + 혜택 중심
서브헤드:     PAS의 Agitation — 고통 증폭
스토리:       StoryBrand SB7 — 고객이 주인공
증거 섹션:    사회적 증거 + 권위 (숫자, 후기)

## 14. 소비자 행동 모델 진화

> AIDA(1898) → AIDMA(1920s) → AISAS(2005) → 주언규 5단계(현재)

### AIDA — Elias St. Elmo Lewis (1898)
```
Attention (주의) → Interest (관심) → Desire (욕구) → Action (행동)
```
- 광고·인쇄매체 시대 모델
- 단방향 커뮤니케이션 전제
- **서비스 적용**: 썸네일(Attention) → 제목(Interest) → 영상 내용(Desire) → CTA(Action)

### AIDMA — Samuel Roland Hall (1920s)
```
Attention → Interest → Desire → Memory (기억) → Action
```
- TV·라디오 시대 추가 요소: Memory (브랜드 기억)
- **서비스 적용**: 브랜딩 요소 반복으로 기억에 남기기

### AISAS — 덴츠 (2005, 인터넷 시대)
```
Attention → Interest → Search (검색) → Action → Share (공유)
```
- 인터넷 시대 핵심 변화: **검색** + **공유** 추가
- 소비자가 능동적으로 정보를 찾고 경험을 나눔
- **서비스 적용**:
  - Search: "어떻게 검색할까?" → SEO 최적화 제목
  - Share: "왜 공유할까?" → 감정적 트리거 + 실용 가치

### 주언규 소비자 행동 5단계 (유튜브 특화)
```
현상 → 욕구 → 계획 → 행동 → 보상
```
- **현상**: 일상에서 불편함/문제를 인식
- **욕구**: 해결하고 싶다는 욕구 발생
- **계획**: 어떻게 해결할지 방법 탐색 (유튜브 검색)
- **행동**: 솔루션 구매/실행
- **보상**: 결과 경험, 공유
- **키 콘텐츠 법칙**: 반드시 **현상**에서 출발해야 한다 (제품 특징에서 출발 X)

### 모델 진화 비교표

| 모델 | 시대 | 핵심 추가 개념 | 콘텐츠 전략 |
|------|------|--------------|------------|
| AIDA | 인쇄 시대 | 주의→행동 선형 | 광고 카피 |
| AIDMA | 방송 시대 | 기억(Memory) | 브랜딩 반복 |
| AISAS | 인터넷 | 검색+공유 | SEO + 바이럴 |
| 주언규 5단계 | 유튜브 | 현상→보상 | 키/풀링 콘텐츠 분리 |

## 15. Jobs-To-Be-Done

> 출처: Clayton Christensen, *The Innovator's Dilemma* + *Competing Against Luck* (2016)

### 핵심 원칙
**"사람들은 제품을 구매하는 게 아니라, 특정 '일(Job)'을 완수하기 위해 제품을 '고용(Hire)'한다"**

### 밀크셰이크 실험 (Christensen의 대표 사례)
- 맥도날드가 밀크셰이크 판매를 늘리려 했으나 개선 실패
- 분석 결과: 아침 출근길 고객이 "운전 중 한 손으로 먹을 수 있고, 배부르게 채워주는 무언가"로 밀크셰이크를 '고용'하고 있었음
- **인사이트**: 경쟁자는 다른 밀크셰이크가 아니라 바나나, 베이글이었다

### JTBD 3가지 차원

| 차원 | 설명 | 유튜브 적용 |
|------|------|------------|
| Functional Job | 실제 기능적 목표 | "유튜브로 월 100만원 벌기" |
| Emotional Job | 감정적 만족 | "불안하지 않고 싶다", "인정받고 싶다" |
| Social Job | 사회적 이미지 | "성공한 크리에이터로 보이고 싶다" |

### 서비스 적용
```
사용자가 이 서비스를 '고용'하는 이유:
- Functional: 아이템 선정 + 수익화 방법을 빠르게 알고 싶다
- Emotional: 혼자 다 알아봐야 한다는 불안감 해소
- Social: 정보력 있는 크리에이터로 느끼고 싶다

→ 서비스 카피에 세 가지 모두 반영해야 전환율 최대화
```

### JTBD 기반 제목 공식
```
❌ 기능 중심: "AI가 아이템을 추천해드립니다"
✅ Job 중심: "6개월 고민하던 아이템, 3분 만에 결정하세요"
         (Functional Job: 빠른 결정 + Emotional Job: 불안 해소)
```

## 16. Hook Model

> 출처: Nir Eyal, *Hooked: How to Build Habit-Forming Products* (2014)

### 4단계 순환 사이클

```
Trigger (트리거)
    ↓
Action (행동)
    ↓
Variable Reward (가변 보상)
    ↓
Investment (투자)
    ↓
[다시 Trigger로]
```

### 1단계: Trigger (트리거)

| 유형 | 설명 | 예시 |
|------|------|------|
| External (외부) | 외부 자극 — 알림, 광고, 친구 추천 | 유튜브 추천 알고리즘 노출 |
| Internal (내부) | 감정·습관 — 불안, 지루함, FOMO | "오늘도 수익 없을까봐" 불안 |

**핵심**: 외부 트리거로 시작해 내부 트리거로 발전시켜야 습관화

### 2단계: Action (행동)
**포그 행동 모델 (BJ Fogg)**: Behavior = Motivation × Ability × Trigger
- 동기(Motivation)와 능력(Ability)이 동시에 충족될 때만 행동 발생
- **서비스 적용**: 클릭 1번으로 결과 나오게 → Ability 최대화

### 3단계: Variable Reward (가변 보상)
**슬롯머신 원리**: 예측 불가능한 보상이 중독성 만든다
- **보상 유형**:
  - Rewards of the Tribe (사회적): 좋아요, 댓글, 인정
  - Rewards of the Hunt (사냥): 새로운 정보, 발견
  - Rewards of the Self (자아): 성취감, 완료감
- **서비스 적용**: 시장성 점수 결과가 매번 다르게 느껴지도록 설계

### 4단계: Investment (투자)
사용자가 시간·데이터·노력을 투입할수록 서비스를 떠나기 어려워짐
- **서비스 적용**: 프로필 입력 → 아이템 기록 → 진행률 저장 → 이탈 방지

### 유튜브 채널에 Hook Model 적용
```
Trigger:  썸네일+제목 (외부) → 성장에 대한 불안감 (내부)
Action:   클릭 1번으로 시청
Variable Reward: 영상마다 다른 인사이트 (Hunt)
Investment: 구독 → 댓글 → 커뮤니티 참여 → 이탈 불가
```

## 17. Pull vs Push 마케팅

### 개념 정의

| | Push 마케팅 | Pull 마케팅 |
|--|------------|------------|
| 방향 | 브랜드→소비자 강제 노출 | 소비자가 스스로 찾아옴 |
| 예시 | TV 광고, 배너, 콜드메일 | SEO, 유튜브, 블로그 |
| 비용 | 지속 비용 발생 | 초기 투자 후 복리 효과 |
| 신뢰도 | 낮음 (광고 피로감) | 높음 (내가 찾은 정보) |
| 전환율 | 낮음 (1-3%) | 높음 (10-30%) |

### 유튜브 = 최강의 Pull 채널
```
검색 Pull:   유튜브는 세계 2위 검색엔진 (Google 다음)
추천 Pull:   알고리즘이 관심사 기반으로 자동 연결
장기 Pull:   에버그린 콘텐츠는 수년간 구매 의도자 유입
```

### Pull 마케팅 깔때기 (유튜브 특화)
```
전체 시청자 (관심사 일치)
    ↓ 구독
팬 (정기 시청자)
    ↓ 키 콘텐츠 시청
잠재 소비자 (구매 고려)
    ↓ 랜딩페이지/링크 클릭
구매자
    ↓ 만족
재구매자 + 추천인 (가장 강력한 Pull)
```

### 풀링 콘텐츠 = Pull 깔때기 상단
- **목적**: 전체 시청자 → 잠재 소비자 전환
- **키 콘텐츠**: 잠재 소비자 → 구매자 전환
- **구별 질문**: "이 영상 보고 살 것 같아?" (키) vs "키콘텐츠 볼 것 같아?" (풀링)

### 서비스 적용
```
서비스 자체도 Pull 전략:
- 무료 시장성 점수 → 유료 전환 (Reciprocity + Pull)
- 가치 있는 무료 기능 → 스스로 공유하게 만듦
- 사용자가 결과를 SNS 공유 → 자동 바이럴 Pull
```

## 18. CRO 전환율 최적화

> CRO = Conversion Rate Optimization
> 출처: Nielsen Norman Group, Baymard Institute, ConversionXL 연구 종합

### 50ms 법칙 — 첫인상의 과학
- 사용자는 **50밀리초(0.05초)** 안에 웹페이지 첫인상 형성
- 이후 판단은 그 첫인상을 **정당화**하는 방향으로 진행 (확증 편향)
- **서비스 적용**: 랜딩 화면 첫 뷰포트가 전부. 스크롤 전에 설득 완료해야 함

### CRO 핵심 심리 원칙 10가지

| 원칙 | 내용 | 서비스 적용 |
|------|------|------------|
| 명확성 | 5초 내에 "이게 뭔지" 알아야 함 | 헤드라인에 서비스 핵심 1줄 |
| 인지 부하 감소 | 선택지 줄이기, 단순화 | 버튼 1개, 질문 최소화 |
| 사회적 증거 | 숫자+사진+이름 조합이 최강 | "OOO명이 사용 중" + 실제 후기 |
| 진행 표시 | 진행률 보이면 완료율 +68% | 단계 표시 바 |
| 손실 회피 | "놓치면" 프레임 | "지금 안 하면 경쟁자에게 뒤처짐" |
| 앵커링 | 먼저 큰 수 제시 후 작은 수 | 원래 가격 → 할인 가격 |
| 호기심 갭 | 정보 일부만 공개 | 결과 미리보기 블러 처리 |
| 희소성 | 한정성 강조 | "이 카테고리 진입자 급증 중" |
| 권위 | 숫자+전문성 지표 | 분석 데이터 기반 수치 제시 |
| 일관성 | 작은 행동 먼저 | 이메일 입력 → 결과 확인 → 가입 |

### F패턴 & Z패턴 — 눈동자 추적 연구

```
F패턴 (텍스트 중심 페이지):
→ 상단 가로 읽기 (헤드라인)
→ 두 번째 가로 읽기 (서브헤드)
→ 왼쪽 세로 스캔 (불릿 포인트)

Z패턴 (이미지 중심 페이지):
→ 좌상단 (로고/브랜드)
→ 우상단 (네비게이션/CTA)
→ 좌하단 (설명)
→ 우하단 (최종 CTA)

서비스 적용: 핵심 CTA는 항상 오른쪽 하단 또는 중앙 상단
```

### 버튼 CTA 최적화
```
❌ "클릭하세요", "확인", "제출"
✅ "내 아이템 시장성 무료 분석하기"
✅ "3분 만에 수익화 아이템 찾기"
✅ "지금 바로 분석 시작 →"

원칙: 동사 + 혜택 + 긴박감
```

## 19. 유튜브 알고리즘 2025

> 출처: YouTube Creator Academy, VidIQ/TubeBuddy 분석, 크리에이터 이코노미 리포트 2025

### 알고리즘 핵심 지표 2025

| 지표 | 가중치 | 설명 |
|------|--------|------|
| CTR (클릭률) | ★★★★★ | 썸네일+제목이 얼마나 클릭 유발하는가 |
| AVD (평균 시청 시간) | ★★★★★ | 절대 시간 (분 단위) |
| 시청 완료율 | ★★★★☆ | 영상 끝까지 본 비율 |
| 재시청률 | ★★★★☆ | 다시 찾아보는 비율 |
| 저장률 | ★★★☆☆ | 나중에 보기 저장 |
| 공유율 | ★★★☆☆ | 외부 확산 |
| 좋아요/싫어요 비율 | ★★☆☆☆ | 참고 지표 |
| 댓글 | ★★☆☆☆ | 참여도 신호 |

### CTR 벤치마크 (2025 기준)
```
4% 미만:   개선 필요 (썸네일/제목 재작업)
4-8%:      평균 (유지)
8-12%:     양호
12%+:      우수 (알고리즘 적극 추천)
```

### 평균 시청 시간 최적화
```
첫 30초: 훅 → 다음 볼 이유 제시 (시청 유지율 결정)
2-3분:   핵심 가치 제공 (이탈 방지 포인트)
중반:    패턴 인터럽트 (화면 전환, 예시, B-roll)
마지막:  CTA + 다음 영상 예고 (연속 시청 유도)
```

### 2025 알고리즘 변화 포인트
1. **홈피드 > 검색**: 추천 알고리즘 비중 상승, 검색 의존도 감소
2. **Shorts 분리**: Shorts는 별도 알고리즘, 롱폼과 교차 영향 제한
3. **주제 일관성**: 채널 주제 일관성 높을수록 구독자 세그먼트 정밀화
4. **시청 이력 반영**: 같은 주제 영상 연속 추천 → 에버그린 콘텐츠 장기 노출
5. **클릭+이탈 패널티**: CTR 높아도 즉시 이탈하면 오히려 추천 감소

### 업로드 최적 전략
```
주 2-3회 > 매일 (품질 > 양)
일관된 시간대 (구독자 습관화)
첫 48시간이 전체 성과의 70% 결정
→ 업로드 직후 고정댓글+SNS 공유 필수
```

### 서비스 적용
```
서비스가 생성하는 콘텐츠 출력물에 적용:
- 제목: CTR 12%+ 타깃 → 호기심갭 + 숫자 + 손실회피 조합
- 대본: 첫 30초 훅 공식 자동 적용
- 썸네일 문구: 3-5단어, 대비색, 감정 얼굴 권장
- 업로드 시간: 타깃 구독자 활성 시간대 추천
```

## 20. 에버그린 콘텐츠 전략

> Evergreen Content = 시간이 지나도 가치가 유지되는 콘텐츠

### 에버그린 vs 시즌성 비교

| | 에버그린 | 시즌성 | 데일리 | 히어로 |
|--|---------|--------|--------|--------|
| 수명 | 수년 | 수주-수개월 | 1-3일 | 수개월 |
| ROI | 복리 증가 | 단기 급상승 후 소멸 | 낮음 | 높음(불안정) |
| 제작 난이도 | 중-높음 | 중 | 낮음 | 매우 높음 |
| 예시 | "유튜브 시작 방법" | "2025 트렌드" | 일상 vlog | 콜라보/이벤트 |

**주언규 전략**: 에버그린 + 데일리 조합 = 안정적 성장

### 에버그린 콘텐츠 특징
```
✅ 검색 가능한 키워드 기반 (How to, 방법, 이유)
✅ 시간·트렌드 초월하는 문제 해결
✅ 수개월 후에도 검색해서 찾아오는 콘텐츠
✅ 1년에 1-2회 업데이트로 상위 노출 유지
✅ 클래식 주제 (기초, 입문, 완전 가이드)
```

### 에버그린 제목 패턴
```
[타깃] + [문제/목표] + [솔루션 암시]

예시:
"유튜브 처음 시작하는 분들을 위한 완전 가이드"
"구독자 없어도 수익 내는 방법 (2025년 검증)"
"아이템 선정부터 첫 판매까지 — 실전 로드맵"
```

### 뷰트랩 활용 에버그린 발굴 (주언규 방식)
```
1. 성장 영상 찾기 (조회수/구독자 비율 높은 영상)
2. 같은 맥락 더 좋은 표현 찾기
3. 노출 확률 높은 키워드 선정
4. 클래식 주제 확인 (5년 전에도 있던 주제인가?)
5. 떡상 채널 추적 (급성장 채널의 에버그린 발굴)
```

### 에버그린 콘텐츠 업데이트 전략
```
6개월마다: 데이터/통계 수치 업데이트
1년마다: 제목에 연도 추가 재업로드
트렌드 변화 시: 해당 섹션만 수정, 전체 리라이팅 X
```

## 21. Customer Journey Map

> 소비자가 인지 → 구매 → 재구매까지 거치는 전체 경험 지도

### 7단계 여정 (디지털 시대)

| 단계 | 소비자 상태 | 유튜브 접점 | 서비스 적용 |
|------|------------|------------|------------|
| 1. 무인식 | 문제 없음 | 풀링 콘텐츠 | 일상 공감 영상 |
| 2. 문제 인식 | 불편함 느낌 | 풀링 콘텐츠 | 현상 공감 썸네일 |
| 3. 정보 탐색 | 해결책 검색 | 키 콘텐츠 | SEO 제목, 에버그린 |
| 4. 대안 평가 | 여러 솔루션 비교 | 키 콘텐츠+댓글 | 사회적 증거+권위 |
| 5. 구매 결정 | 선택 | 랜딩페이지/링크 | CTA+희소성 |
| 6. 구매 후 경험 | 결과 평가 | 후기 영상 | 보상 확인 콘텐츠 |
| 7. 재구매/추천 | 팬 전환 | 커뮤니티 | 커뮤니티+멤버십 |

### 각 단계별 콘텐츠 전략

**단계 1-2 (풀링 콘텐츠 영역)**
- 제품 언급 X, 문제 공감만
- 구별법: "이 영상 보고 키콘텐츠 볼 것 같아?"
- 예: "직장 다니면서 투잡 못 하는 이유"

**단계 3-4 (키 콘텐츠 영역)**
- 솔루션 제시, 제품 자연스럽게 등장
- 구별법: "이 영상 보고 살 것 같아?"
- 예: "직장인이 유튜브로 월 100만원 버는 방법"

**단계 5 (전환 콘텐츠 영역)**
- 랜딩페이지, 링크, 설명란
- 사회적 증거 + CTA

### 감정 여정 (Emotional Journey)
```
무인식:  평온 → 문제 인식: 불편/불안 → 탐색: 기대/혼란
→ 평가: 의심/신뢰 → 구매: 설레임/불안 → 경험: 만족/실망
→ 재구매: 충성심/전도사
```
**핵심**: 각 단계의 **감정**을 맞춰주는 콘텐츠 = 전환율 극대화

## 22. Product-Market Fit

> 출처: Steve Blank (Customer Development), Eric Ries (Lean Startup), Marc Andreessen (PMF 정의)

### Marc Andreessen의 PMF 정의
**"시장이 제품을 끌어당기는 느낌 (Product-Market Fit = 당기는 힘)"**
- PMF 전: 아무리 밀어도 안 됨 (Push)
- PMF 후: 홍보 없어도 퍼짐 (Pull)

### PMF 측정 방법 — Sean Ellis 테스트
```
질문: "이 서비스가 없어진다면 어떻게 느낄 것 같나요?"
□ 매우 실망할 것 같다
□ 약간 실망할 것 같다
□ 별로 실망하지 않을 것 같다

기준: "매우 실망" 응답 ≥ 40% = PMF 달성
```

### Steve Blank Customer Discovery 4단계
```
1단계: Customer Discovery — 가설 수립 + 고객 인터뷰
2단계: Customer Validation — 반복 가능한 판매 검증
3단계: Customer Creation — 수요 창출 본격화
4단계: Company Building — 조직화
```

### Lean Startup 핵심 사이클
```
Build (만들기) → Measure (측정) → Learn (학습) → [반복]
```
- **MVP**: 최소 기능 제품으로 빠르게 검증
- **피벗**: 데이터 기반 방향 전환

### 유튜브 아이템 PMF 검증 공식 (주언규식)
```
1. 키 콘텐츠 1개 올리기
2. 조회수 1,000 당 구매 문의 측정
3. 댓글에 "어디서 사요?"가 나오면 PMF 신호
4. 전환율 1% 이상 = PMF 달성 기준

뷰트랩 PMF 검증:
- 이미 팔리고 있는 아이템의 성장 영상 확인
- "이미 시장이 원하고 있음"이 증명된 아이템 선택
```

### 서비스 내 시장성 점수 = PMF 사전 검증 도구
```
시장성 점수 구성 요소:
- 검색량 (수요 존재 여부)
- 경쟁 강도 (진입 가능성)
- 수익화 사례 (PMF 달성 선례)
- 성장 트렌드 (시장 방향성)
```

## 23. 카피라이팅 제목 공식 고급

> 기존 AIDA/PAS/BAB 외 제목 특화 고급 공식

### 호기심 갭 (Curiosity Gap) — George Loewenstein (1994)
**"알고 싶은 것과 아는 것 사이의 간격이 클수록 클릭 욕구 증가"**
```
공식: [관련성 있는 주제] + [부분 정보 공개] + [핵심 누락]

예시:
"이 한 가지만 바꿨더니 조회수가 10배 됐습니다" (뭔지 안 말함)
"3개월 만에 구독자 10만 된 채널의 공통점" (공통점 안 말함)
"대부분이 모르는 유튜브 수익화 비밀" (비밀 안 말함)
```

### 숫자의 심리학
```
홀수 > 짝수: "7가지" > "8가지" (더 구체적으로 느껴짐)
구체성: "327%" > "300%" (신뢰도 상승)
대비: "3분 만에" > "빠르게" (시간 구체화)
규모: "1만 명이" > "많은 사람이" (사회적 증거 강화)
```

### 손실 회피 제목 공식
```
공식: "지금 [행동] 안 하면 [손실] 당합니다"

예시:
"이 아이템 모르면 내년에 후회합니다"
"지금 유튜브 시작 안 하면 늦는 이유"
"2025년 안에 시작 못 하면 경쟁에서 밀립니다"

손실 회피 = 동일 이익의 2.5배 효과 (Kahneman)
```

### A이지만 B 대비 구조
```
공식: "[기대 반전] + [놀라운 결과]"

예시:
"구독자 1000명인데 월 300만원 버는 방법"
"영상 안 올리고 수익화하는 채널의 비밀"
"광고 없이 팔리는 제품 만드는 법"
```

### 타깃 명시 공식
```
공식: "[정확한 타깃 명시] + [그들의 문제/목표]"

예시:
"직장 다니면서 유튜브 하는 분들만 보세요"
"구독자 1000명 미만인 채널 운영자 필독"
"40대 이후 처음 유튜브 시작하는 분들"
```

### 제목 A/B 테스트 기준
```
클릭 후 30초 이내 이탈률: 좋은 제목인지 측정
CTR × 시청유지율 = 실제 알고리즘 점수
A/B 테스트 최소 샘플: 영상당 500뷰 이상
```

## 24. 한국 인플루언서 마케팅 데이터

> 출처: 오픈서베이 소셜미디어 리포트 2024, DMC미디어, 한국 MCN 협회 데이터

### 한국 유튜브 소비 행동 2024-2026

| 지표 | 데이터 |
|------|--------|
| 유튜브 월간 이용자 | 4,600만 명 이상 (전 국민의 90%) |
| 일평균 유튜브 이용 시간 | 약 40-45분 |
| 구매 결정에 유튜버 영향 | 35-45% (20-30대) |
| 구독자보다 조회수 신뢰 | 68% |
| 쇼츠로 첫 발견 후 롱폼 시청 | 42% |

### 한국 소비자 구매 결정 흐름
```
유튜브 영상 발견 (40%)
    → 댓글/후기 확인 (78%)
    → 네이버 검색 (65%)
    → 쿠팡/스마트스토어 확인 (55%)
    → 구매 결정
```
**핵심**: 유튜브는 탐색 시작점, 네이버는 신뢰 확인, 쿠팡은 구매 완료

### 카테고리별 인플루언서 마케팅 효과

| 카테고리 | ROI | 전환율 | 특징 |
|---------|-----|--------|------|
| 뷰티/헬스 | 8-12x | 5-8% | 체험 후기 중심 |
| 식품/음료 | 5-8x | 3-5% | 먹방+ASMR 강세 |
| 교육/강의 | 10-20x | 2-4% | 신뢰도+권위 중요 |
| 디지털 제품 | 15-30x | 3-6% | 낮은 원가, 높은 마진 |
| 생활용품 | 4-6x | 2-3% | 실용성 강조 |

### 한국 MZ 세대 콘텐츠 반응 특성
```
✅ 반응 높은 요소:
- 진정성 (꾸미지 않은 솔직함)
- 공감 (내 이야기 같은 느낌)
- 실용 정보 (바로 써먹을 수 있는)
- 유머 (가볍게 웃으며 정보 흡수)
- 소속감 (우리끼리 아는 것)

❌ 반응 낮은 요소:
- 과장된 광고 느낌
- 지나치게 완벽한 연출
- 일방적인 판매 압박
- 트렌드와 동떨어진 표현
```

### 한국 특화 구매 심리
```
체면 문화: "이걸 사면 어떻게 보일까" → 사회적 증거 극대화 효과
눈치 문화: 유행이면 따름 → 트렌드 + 사회적 증거 결합
빨리빨리: 즉각적 결과 원함 → "3분 만에", "바로 확인" 카피 효과
가성비: 가격 대비 효용 중시 → 가치 증명 필수
커뮤니티: 카카오 단톡방 추천 강력 → 고정댓글 공유 유도
```

### 넥스트 트렌드 2025-2026
```
AI 크리에이터 경제: AI 도구 활용 크리에이터 급증 → 차별화 어려워짐
→ 대응: 개인 경험 + 스토리 강화 (AI가 못 하는 것)

발견형 쇼핑 (Discovery Commerce): 필요 없어도 발견하면 삼
→ 대응: 풀링 콘텐츠로 상단 넓히기

초개인화: 나만을 위한 맞춤 느낌 원함
→ 대응: "직장인", "30대 여성", "초보" 등 타깃 명시
```

## 27. TikTok 알고리즘 2026

> 출처: Buffer TikTok Algorithm Guide 2026, Hootsuite, Socialync, BeatsToRapon

### FYP (For You Page) 핵심 작동 원리

**2026 최대 변화**: 팔로워 기반 → **소주제(Micro-Niche) 기반** 배포
```
1단계: 소규모 테스트 (팔로워 일부에게 먼저 노출)
2단계: 팔로워 반응 분석 (완료율 70%+ = 확산 신호)
3단계: 같은 관심사 비팔로워에게 확산
4단계: 글로벌 FYP 노출 (성과 높을 때)
```

### TikTok 핵심 지표 2026

| 지표 | 기준 | 의미 |
|------|------|------|
| 완료율 | 70%+ 목표 | FYP 확산 트리거 (2024: 50% → 2026: 70%) |
| 인트로 리텐션 | 첫 3초 70%+ | "intro retention" 별도 측정 |
| 공유율 | 벤치마크 이상 | 외부 확산 = 최강 신호 |
| 저장률 | 높을수록 좋음 | 에버그린 콘텐츠 지표 |
| 팔로우 전환 | 시청 후 팔로우 | 채널 성장 지표 |

### 2026 TikTok 알고리즘 변화 5가지
```
1. 완료율 기준 상향: 50% → 70% (더 강한 훅 필요)
2. 팔로워 우선 배포: 내 팔로워가 안 보면 확산 X
3. 키워드 검색 강화: TikTok SEO — 캡션에 검색 키워드 필수
4. 시리즈 콘텐츠 우대: 연속 시청 유도 시리즈 형식 알고리즘 가산점
5. 진정성 신호: 고품질 프로덕션보다 자연스러운 영상 선호
```

### TikTok 훅 공식 (완료율 70%+ 달성)
```
유형 1: 대담한 주장 (Bold Claim)
  "이 방법 모르면 유튜브 절대 안 됩니다"

유형 2: 놀라운 사실 (Surprising Fact)
  "구독자 100명인 채널이 1만명 채널보다 돈 더 버는 이유"

유형 3: 시각적 패턴 인터럽트 (Visual Pattern)
  처음 1프레임에 결과 화면 (통장, 성과)

유형 4: 즉각 가치 제공
  "5초 안에 [문제] 해결법 알려드림"

유형 5: 논란성 의견 (Controversial Take)
  "조회수 높다고 수익화 잘 된다는 건 거짓말입니다"
```

### TikTok 최적 업로드 전략
```
빈도: 주 3-5회 (일관성 > 양)
시간: 타깃 깨어있는 시간대 (한국: 오전 7-9시, 밤 9-11시)
길이: 15-60초 (완료율 최적화)
캡션: 검색 키워드 포함 (TikTok SEO)
댓글 응답: 업로드 후 1시간 내 모든 댓글 응답 (알고리즘 가산)
```

### 서비스 연결 (TikTok 출력물)
```
- 15-60초 대본 (완료율 70% 타깃 구조)
- 첫 3초 훅 문장
- 캡션 + 검색 키워드 (TikTok SEO)
- 시리즈 기획 ("파트 1/파트 2" 연속 구조)
```

- 여러 주제: 잡블로그 판정 → 노출 감소

```

신뢰성: 출처 명시, 전문 용어 적절 사용

독창성: 복붙/중복 콘텐츠 패널티
```

**네이버 블로그 상위노출 공식 (2026)**
```
1. 주제 집중: 1개 카테고리만 (예: "유튜브 수익화")
2. 포스팅 주기: 주 3회 이상 꾸준히
3. 길이: 1,500-2,500자 + 이미지 5개 이상
4. 키워드: 제목에 핵심 키워드, 본문에 자연스럽게
5. 체류시간: 읽는 데 3-5분 걸리는 분량
6. 이미지: 원본 촬영 이미지 (복붙 이미지 패널티)
```

```

        ↓

        ↓
[고정댓글 + 링크바이오 → 서비스 랜딩페이지]

**서비스 적용**: 사용자가 아이템 1개 선택하면 → **6개 플랫폼 출력물 동시 생성**

## 41. 인지편향 마스터 — 27가지 핵심

> 출처: Kahneman, Fogg, Convertize, VentureHarbour (03-01 파일)

### 가격/가치 판단 (5개)
```
앵커링: 첫 숫자가 기준 → 가격 인식 32% 향상
미끼효과: 제3선택지 추가 → 프리미엄 40-60% 증가
차임프라이싱: $9.99 < $10 인식 → 판매 30% 증가
프레스티지: 고가 = 고품질 (프리미엄 시장)
번들: 묶음 = 절약 인식 → 인식 가치 총합 > 실제 합산
```

### 손실/위험 회피 (5개)
```
손실 회피: 손실 고통 = 이득 기쁨 × 2.5배 → 잃는 것 프레이밍
현상 유지: 기본값이 선택됨 → 원하는 옵션을 기본값으로
매몰 비용: 이미 투자한 것 때문에 계속함 → 무료 체험 후 전환
소유 효과: 내 것은 더 가치 있음 → 개인화 데이터 쌓기
FOMO: 놓치는 것에 대한 두려움 → 실시간 활동 표시
```

### 사회적 판단 (5개)
```
사회적 증거: 다수 행동 따름 → 숫자+후기+실시간 표시
권위 편향: 전문가 말 신뢰 → 데이터/수상/언론 활용
희소성: 희귀할수록 원함 → 진짜 한정 수량/기간
확인 편향: 믿는 것 확인 → 타깃의 세계관에서 출발
후광 효과: 한 긍정 = 전체 긍정 → 첫인상 완벽하게
```

### 선택/정보 처리 (5개)
```
선택 과부하: 선택 많을수록 결정 못 함 → 플랜 3개 이하
호기심 갭: 조금 알 때 더 궁금 → "3가지 이유가 있습니다"
노출 효과: 자주 볼수록 호감 → 7번 접촉 법칙
처리 유창성: 쉬운 것이 더 진실 → 단순한 문장
구체성: 숫자/사례가 설득력 → "37명이 달성"
```

### 시간/기타 (7개)
```
즉시 보상: 지금 당장 선호 → "즉시 다운로드"
계획 오류: 시간 과소평가 → "혼자 100시간 vs AI 10분"
미래 자아 불연속: 미래의 나 = 낯선 사람 → 미래 페이싱
절정-종단: 절정+마지막으로 경험 기억 → 첫 성공 + 이탈 시 긍정
프레이밍: 표현 방식이 판단 변경 → "90% 성공률" vs "10% 실패"
더닝-크루거: 초보=자신감 높음 → 입문 쉬운 것처럼 포지셔닝
바넘 효과: 보편적 = 개인적으로 느낌 → "당신 같은 크리에이터"
```
