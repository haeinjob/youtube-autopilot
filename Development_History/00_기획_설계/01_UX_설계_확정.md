## 1. 전체 유저 플로우

```
[랜딩페이지]
     │
     ▼
[구글/카카오 로그인] ──────────────────────────────────────────┐
     │                                                         │ (비로그인)
     ▼                                                         ▼
[시작 화면: "어떻게 시작할까요?"]                    [체험판 - 결과 일부만 공개]
     │
     ├── A. 내 정보 입력 → 맞춤 아이템 추천
     ├── B. 아이템 직접 입력 → 시장성 검증
     ├── C. 내 유튜브 채널 URL → 채널 진단
     └── D. 아무것도 모르겠어요 → 지금 제일 잘 팔리는 상품 3개 랜덤 추천
     │
     ▼
[마법사: 단계별 입력] (2~3단계, 경로마다 다름)
     │
     ▼
[결과 대시보드] ← 모든 경로의 최종 목적지
```

---

## 2. 진입 방식 4가지 상세

### A. 내 정보로 맞춤 추천
**대상**: 뭘 팔지 모르지만 내 정보로 AI 추천받고 싶은 사람

**입력 단계**:
1. 나이대 선택 (20대 / 30대 / 40대 / 50대+)
2. 성별 선택
3. 관심사·취미 선택 (복수 선택 가능, 태그 UI)
   - 예: 운동, 요리, 패션, 육아, 재테크, IT, 여행, 반려동물...
4. 직업 입력 (선택사항, 텍스트)
5. 월 목표 매출 선택 (100만 / 300만 / 500만 / 1000만+)

**출력**: AI가 아이템 5개 추천 + 각각 "왜 나한테 맞는지" 근거
→ 하나 선택 → 결과 대시보드

---

### B. 아이템 직접 입력
**대상**: 팔고 싶은 게 이미 있는 사람

**입력 단계**:
1. 아이템/키워드 텍스트 입력
2. 카테고리 확인 (AI 자동 분류, 수정 가능)
3. 월 목표 매출 선택

**출력**: 시장성 점수 즉시 → 결과 대시보드

---

### C. 채널 URL 진단
**대상**: 유튜브 채널이 이미 있는 사람

**입력 단계**:
1. 유튜브 채널 URL 입력
2. 현재 팔고 있는 것 또는 팔고 싶은 것 입력 (선택사항)
3. 월 목표 매출 선택

**출력**: 채널 현황 분석 (구독자, 평균 조회수) + 수익화 갭 진단 → 결과 대시보드

---

### D. 완전 모르겠어요 (랜덤 추천)
**대상**: 아무것도 모르는 완전 초보, 빠르게 시작하고 싶은 사람

**입력 단계**:
1. 나이대 + 성별 선택 (딱 2개만)

**출력**: 지금 이 연령/성별에서 제일 잘 팔리는 상품 3개 랜덤 추천 카드
→ 하나 선택 → 결과 대시보드

**UX 포인트**: "지금 OO대 여성에게 가장 잘 팔리는 상품 TOP 3" 형태로 표시
→ 선택 장벽 최소화, 즉시 시작 가능

### 3.2 각 섹션 공통 기능

| 버튼 | 기능 |
|------|------|
| **클릭해서 펼치기** | 아코디언 방식으로 내용 표시/숨기기 |
| **📋 복사** | 텍스트 클립보드 복사 (Copy 버튼, 각 텍스트 블록마다) |
| **🔄 다시 생성** | 해당 섹션만 AI 재생성 (다른 섹션은 유지) |
| **💾 저장** | 해당 섹션 결과를 내 계정에 영구 저장 |

### 3.3 섹션별 출력 내용

#### ① 시장성 점수
- 종합 점수 (100점 만점)
- PASS / CONDITIONAL / FAIL 판정
- 세부 항목: 객단가, 시장 규모, 경쟁 포화도, 재구매 가능성
- 개선 제안 (FAIL 시 대안 아이템 3개 제시)

#### ② 카테고리 1등 상품
- 해당 카테고리 상위 판매 상품 3개
- 각각: 상품명, 예상 가격대, 주요 특징

#### ③ 상위 크리에이터
- YouTube API 기반 관련 채널 상위 5개
- 각각: 채널명, 구독자 수, 평균 조회수, 주요 콘텐츠 유형
- "이 사람들이 잘 되는 이유" 한 줄 분석

#### ④ 목표 조회수 계산기
- 월 목표 매출 입력 슬라이더
- 역산 출력: 필요 고객 수 → 랜딩 방문자 → 키 콘텐츠 조회수 → 풀링 조회수 → **하루 필요 조회수**
- "지금 내 채널 평균 조회수 입력" → 달성 예상 기간 계산

#### ⑤ 유튜브 제목 10개
- 풀링 콘텐츠용 5개 (조회수 극대화)
- 키 콘텐츠용 5개 (구매 전환 목적)
- 각 제목 옆 Copy 버튼
- 제목 공식 태그 표시 (예: "숫자+손실회피형", "비교대비형")

#### ⑥ 유튜브 대본
- 훅 (0~15초): 충격/공감/질문 3가지 버전
- 본론: 핵심 메시지 + 증거 + 비유
- 클로징: CTA + 희소성 요소
- 전체 복사 버튼 + 섹션별 복사 버튼

#### ⑦ 랜딩페이지 카피
- 헤드라인 (결과 중심)
- 공감 서사 (고통 → 변화 가능성)
- 증거 섹션 구조
- 제품 소개 (혜택 중심)
- CTA 버튼 문구 3개 옵션

#### ⑧ 고정 댓글 템플릿
- 핵심 요약 1줄
- 무료 자료/링크 안내 형식
- 다음 영상 예고 문구
- 참여 유도 질문

## 4. 로그인 & 저장 구조

### 4.1 로그인
- 구글 OAuth / 카카오 로그인
- 비로그인: 결과 화면 진입 가능, 단 텍스트 복사 3회 제한 + 저장 불가
- 로그인: 전체 무제한 (구독 플랜에 따라)

### 4.2 저장 구조
```
내 계정
├── 내 아이템 목록 (저장한 아이템들)
│   ├── [아이템명] — 저장일
│   │   ├── 시장성 점수
│   │   ├── 제목 모음
│   │   ├── 대본
│   │   ├── 랜딩 카피
│   │   └── 고정 댓글
│   └── ...
└── 즐겨찾기 (특정 섹션만 저장)
```

## 5. 반응형 & 디자인 방향

- **모바일 우선** (대부분 스마트폰으로 접근 예상)
- 컬러: 진한 네이비 + 포인트 오렌지/옐로우 (신뢰감 + 에너지)
- 폰트: 한국어 Pretendard
- 아이콘: Lucide Icons
- 참고 레퍼런스: Linear(대시보드 밀도), Notion(아코디언 구조), Typeform(마법사 UI)
