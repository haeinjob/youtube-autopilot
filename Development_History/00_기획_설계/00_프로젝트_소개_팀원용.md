## 2. 이 프로젝트는 무엇인가

### 2.1 한 문단 정의

유튜브로 수익화를 꿈꾸는 사람들이 가장 어려워하는 것은 **"무엇을 팔지"를 정하고, 그것을 팔기 위한 콘텐츠를 만드는 것**입니다. 유튜브 오토파일럿은 수백만 원짜리 수익화 강의에서 검증된 알고리즘을 AI가 대신 수행해주는 웹 서비스입니다. 사용자는 자신의 성별·나이·관심사를 입력하거나 팔고 싶은 아이템을 입력하기만 하면, AI가 시장성 검증부터 유튜브 대본·썸네일 문구·랜딩페이지 카피·고정댓글까지 복붙 가능한 형태로 모두 생성해줍니다.

### 2.2 이 프로젝트가 해결하는 문제

유튜브 수익화를 시도하는 사람들에게는 공통된 3가지 벽이 있습니다:

**벽 1. "뭘 팔아야 할지 모르겠다"**
- 유튜브를 시작하면 대부분 "뭘 찍을까"부터 고민한다
- 정작 수익화가 되려면 "뭘 팔까"가 먼저인데, 이 순서를 모른다
- 결국 조회수만 쫓다가 수익화는 영원히 못한다

**벽 2. "시장성 있는 아이템을 어떻게 찾나"**
- 네이버 데이터랩, 유튜브 분석 도구를 다 써봐도 뭘 봐야 하는지 모른다
- 객단가가 낮은 상품을 골라서 아무리 팔아도 수익이 안 난다
- 내 성별·연령대·경험에 맞는 아이템 추천을 해주는 곳이 없다

**벽 3. "콘텐츠를 어떻게 만들어야 클릭이 되나"**
- 영상을 올려도 아무도 안 본다
- 제목과 썸네일이 중요한 건 아는데 어떻게 만들어야 하는지 모른다
- 조회수로 사람을 끌어오는 콘텐츠와 판매로 이어지는 콘텐츠의 차이를 모른다

**유튜브 오토파일럿은 이 3가지 벽을 AI로 허문다.**

### 2.3 이 프로젝트가 '하지 않는 것' (Non-goals)

- ❌ 영상 편집 자동화 (편집 툴이 아님)
- ❌ 유튜브 채널 자동 운영 (AI가 직접 업로드하지 않음)
- ❌ 광고 집행 대행
- ❌ 인플루언서 마케팅 플랫폼
- ❌ 타 SNS(인스타그램, 틱톡) 콘텐츠 생성 — Phase 2 이후 검토

## 3. 왜 이걸 하는가 (시장 기회)

### 3.1 한국 유튜브 크리에이터 시장 규모

| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 유튜브 월간 활성 사용자 | **4,500만명** | 구글 코리아 2024 |
| 수익 창출 채널 수 (구독자 1,000명+) | **약 20만 채널** | 업계 추산 |
| 1인 미디어 시장 규모 | **1조 2,000억원** (2024) | 과기정통부 |
| 수익화 강의 시장 (온라인 강의 전체) | **약 5,000억원** | 에듀테크 업계 추산 |
| 유튜브 수익화 성공률 | **약 5% 미만** | 크리에이터 커뮤니티 통계 |

**핵심**: 95%가 실패하는 시장에, 성공 공식을 AI로 자동화해주는 서비스는 아직 없다.

### 3.2 수익화 강의 시장의 구조적 문제

- 수백만 원짜리 수익화 강의들이 성황 중 (월 수십만 원 코칭 프로그램 다수)
- 강의를 듣고도 실행하지 못하는 사람이 대다수 — **"알아도 못 한다"**
- 강의 내용을 실제로 적용하려면 네이버 데이터랩, 유튜브 분석, 쿠팡 데이터를 수동으로 뒤져야 함
- **이 서비스는 강의를 대신 "실행"해주는 AI다**

### 3.3 AI 도구 시장의 타이밍

- 2024~2026년: 일반인의 AI 도구 사용 급속 확산
- "ChatGPT로 대본 써줘" 단계는 지남 — 이제 "내 비즈니스에 특화된 AI"를 원함
- 수익화에 특화된 AI 도구는 국내에 아직 공백 상태

## 4. 누가 쓰는가 (타겟 유저)

### 4.1 1차 타겟: 유튜브 수익화 초보 (핵심)

**프로필**:
- 25~45세, 직장인 또는 자영업자
- 유튜브 채널은 있거나 시작 예정이지만 수익화는 0원
- "뭘 팔지 모르겠다"는 게 가장 큰 고민
- 수익화 강의 구매 경험 있거나 관심 있음

**이 사람들이 원하는 것**:
- 내 상황에 딱 맞는 아이템 추천
- 시장성 있는 건지 바로 확인
- 제목·대본·카피 직접 안 쓰고 복붙만 하고 싶음

### 4.2 2차 타겟: 1인 사업자·마케터

**프로필**:
- 이미 팔 상품/서비스가 있음
- 유튜브를 마케팅 채널로 활용하고 싶음
- 콘텐츠 제작에 들어가는 시간을 줄이고 싶음

**이 사람들이 원하는 것**:
- 내 상품의 시장성·경쟁 환경 빠른 파악
- 클릭을 유도하는 제목·썸네일 공식
- 판매로 이어지는 랜딩페이지 카피 자동 생성

### 4.3 페인 포인트 요약

| 페인 포인트 | 현재 해결책 | 유튜브 오토파일럿 |
|------------|-----------|-----------------|
| 뭘 팔지 모른다 | 강의 수강 (수백만 원) | AI가 즉시 추천 |
| 시장성 확인이 어렵다 | 뷰트랩·데이터랩 수동 조사 | 점수로 즉시 판정 |
| 제목·대본 못 쓴다 | ChatGPT로 범용 생성 | 수익화 로직 기반 특화 생성 |
| 경쟁자가 뭐 하는지 모른다 | 일일이 유튜브 검색 | 카테고리 1등 + 상위 크리에이터 자동 표시 |
| 랜딩페이지 카피 모른다 | 카피라이터 고용 또는 포기 | 결과→증거→제품 구조로 자동 생성 |

## 5. 무엇을 만드는가 (기능 라인업)

### 5.1 전체 파이프라인

```
[입구 A] 내 정보 입력 (성별/나이/관심사/직업)
         → AI가 아이템 5개 추천 + 각각 "왜 나한테 맞는지" 근거 제시

[입구 B] 아이템 직접 입력 (키워드)
         → 시장성 점수 즉시 출력 (PASS / CONDITIONAL / FAIL)

              ↓ 아이템 선택

[대시보드]
├── 📊 시장 분석
│   ├── 카테고리 내 1등 아이템 실제 데이터
│   ├── 상위 유튜브 크리에이터 목록 (조회수, 구독자, 주요 콘텐츠)
│   └── 목표 조회수 역산 계산기 (매출 목표 입력 → 필요 조회수 출력)
│
├── ✍️ 콘텐츠 생성 (전부 복붙 가능)
│   ├── 유튜브 제목 10개 (풀링용 5개 + 키 콘텐츠용 5개)
│   ├── 썸네일 문구 5개 (3~5단어, 심리 트리거 적용)
│   ├── 유튜브 대본 (훅 15초 + 본론 + 클로징 구조)
│   ├── 쇼츠 대본 (60초 버전)
│   └── 고정 댓글 템플릿
│
└── 🛒 랜딩페이지 카피 생성
    ├── 헤드라인 (결과 중심)
    ├── 공감 서사 (문제 → 변화 가능성)
    ├── 증거 섹션 (사회적 증거 구조)
    ├── 제품 소개 (혜택 중심)
    └── CTA 버튼 문구 (손실 회피 심리 적용)
```

### 5.2 기능별 상세

#### Feature 1: 아이템 추천 (나로부터 출발)
- 입력: 성별, 연령대, 관심사/취미, 직업 (선택)
- 처리: AI가 "이 사람이 진정성 있게 다룰 수 있는 + 객단가 25만원 이상 + 시장 충분한" 아이템 5개 추천
- 출력: 아이템명 + 추천 이유 ("당신의 [경험/취미]에서 출발한 아이템") + 예상 시장성 등급

#### Feature 2: 시장성 검증 스코어
- 입력: 아이템 키워드
- 처리: 객단가 기준 (25만원 이상 여부) + AI가 시장 크기 추정 + 경쟁 포화도 분석
- 출력: 종합 점수 (100점) + PASS/CONDITIONAL/FAIL 판정 + 개선 제안

#### Feature 3: 카테고리 1등 & 상위 크리에이터
- 처리: YouTube Data API로 관련 채널 검색, 조회수/구독자 기준 정렬
- 출력: 상위 5개 채널 카드 (채널명, 구독자수, 평균 조회수, 주요 콘텐츠 유형)

#### Feature 4: 콘텐츠 텍스트 생성기
- 입력: 아이템 + 타겟 감정 + 콘텐츠 유형 선택 (풀링/키)
- 처리: 수익화 알고리즘 기반 제목 공식 적용, 심리 트리거 매핑
- 출력: 즉시 복붙 가능한 텍스트 블록 (Copy 버튼 포함)

#### Feature 5: 랜딩페이지 카피라이터
- 입력: 아이템 + 타겟 고객 고통 + 결과값
- 처리: 결과→증거→제품 구조 자동 적용
- 출력: 섹션별 카피 (헤드라인/서사/증거/제품/CTA) 복붙 가능

## 6. 어떻게 차별화하는가

### 6.1 차별화 4기둥

| # | 기둥 | 구체 내용 | 경쟁 대비 |
|---|------|----------|---------|
| 1 | **수익화 알고리즘 특화** | 범용 AI 글쓰기가 아닌, 검증된 수익화 로직(객단가·결 이론·역산 공식) 기반 생성 | ChatGPT·Jasper = 범용 카피. 이 서비스 = 수익화 전용 |
| 2 | **"나로부터 출발" 개인화** | 성별·나이·직업·취미 입력 → AI가 진정성 있는 아이템 추천. 개인 맞춤형 | 뷰트랩 = 키워드 조회수만. 이 서비스 = 사람 중심 추천 |
| 3 | **전체 파이프라인 통합** | 아이템 선정 → 시장검증 → 콘텐츠 생성 → 랜딩카피까지 한 서비스 내 완결 | 기존 도구들 = 각각 따로 사용해야 함 |
| 4 | **즉시 복붙 실행** | 모든 출력물이 복붙 가능. 생각할 필요 없음 | 강의 = 알려주기만 함. 이 서비스 = 대신 해줌 |

### 6.2 "강의 대신 AI" 포지셔닝

```
기존 방식:
강의 수강 (수백만 원) → 내용 이해 → 직접 적용 → 대부분 실패

유튜브 오토파일럿:
프로필 입력 → AI가 강의 로직 대신 수행 → 복붙만 하면 됨
```

- 기획자는 브라우저에서 결과 확인 후 피드백
