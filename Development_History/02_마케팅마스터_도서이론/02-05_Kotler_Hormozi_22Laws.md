# 마케팅 마스터 도서이론 — Kotler / Hormozi / 22 Laws / Guerrilla

> Philip Kotler / Alex Hormozi / Al Ries & Jack Trout (22 Laws) / Jay Conrad Levinson (Guerrilla)

## 1. Philip Kotler — Marketing Management (1967~현재, 16판까지)

### 핵심 철학
> "마케팅은 수익성 있는 고객 관계를 만들고 유지하는 과정이다."

### STP 프레임워크 (전략의 뼈대)

```
S — Segmentation (시장 세분화)
    인구통계 / 심리분석 / 행동 / 지리적 기준으로 시장 분류

T — Targeting (타겟 선정)
    세분화된 시장 중 가장 매력적인 세그먼트 선택

P — Positioning (포지셔닝)
    타겟 고객의 마음속에 원하는 위치 차지
```

### 4P → 4C → 4E 진화

| 버전 | 요소 | 현대적 해석 |
|------|------|------------|
| **4P (판매자 시각)** | Product / Price / Place / Promotion | 전통적 마케팅 믹스 |
| **4C (고객 시각)** | Customer Value / Cost / Convenience / Communication | 고객 중심 재해석 |
| **4E (경험 시대)** | Experience / Exchange / Everyplace / Evangelism | 디지털 시대 확장 |

### 고객 가치 방정식
```
고객 가치 = (기능적 혜택 + 감정적 혜택 + 사회적 혜택)
            ÷ (금전적 비용 + 시간 비용 + 에너지 비용 + 심리적 비용)

→ 서비스 적용: 가치를 높이거나 비용을 낮추면 구매 결정 쉬워짐
```

### Customer Lifetime Value (CLV) 프레임
```
CLV = 평균 구매가 × 구매 빈도 × 고객 유지 기간

고CLV 고객:
- 최우선 서비스/콘텐츠 제공
- 업셀/크로스셀 타겟
- 커뮤니티 코어 멤버
```

### 디지털 마케팅 4.0 핵심 (Kotler 확장)
```
전통 마케팅 → 디지털 마케팅 변화:
1. 독백 → 대화 (브랜드가 말하기만 → 상호작용)
2. 마케터 중심 → 고객 여정 중심
3. 1:N → N:N (커뮤니티, 바이럴)
4. 단기 판매 → 장기 관계
```

### 서비스 적용
```
콘텐츠 STP 예시:
- Segment: 30대 직장인 / 부업 관심 / 유튜브 초보
- Target: 아직 아이템 못 정한 사람 (가장 넓고 절박함)
- Position: "가장 쉬운 첫 단계 알려주는 채널"
```

## 2. Alex Hormozi — $100M Offers (2021) + $100M Leads (2023)

### 핵심 철학
> "평범한 제품을 거부할 수 없는 오퍼로 포장하면 성공한다."

### Grand Slam Offer 공식 ($100M Offers)

```
거부할 수 없는 오퍼 = 
  드림 아웃컴 × 인지 가능성
  ÷ (시간 지연 × 노력 & 희생)
```

**4가지 레버**
| 레버 | 높이는 방법 | 콘텐츠 적용 |
|------|------------|------------|
| **드림 아웃컴 ↑** | 결과를 극대화해서 표현 | "월 300만원 추가수입" |
| **인지 가능성 ↑** | 증거, 사례, 보증으로 신뢰 구축 | 성공사례 + 데이터 |
| **시간 지연 ↓** | "빠른 결과" 강조 | "3일 안에" "첫 주에" |
| **노력 희생 ↓** | "쉽다" "자동화" 강조 | "버튼 하나로" "AI가 대신" |

### 가치 스택 구조
```
핵심 오퍼
+ 보너스 1 (높은 인지 가치, 낮은 원가)
+ 보너스 2
+ 보너스 3
+ 보증 (리스크 제거)
= 총 가치 > 가격 × 10배 이상

→ "가격이 결정이 아니라 당연한 것"이 되도록
```

### $100M Leads — 리드 생성 4가지 방법

```
1. 워밍 아웃리치 (Warm Outreach)
   → 기존 인맥에게 먼저 접근
   
2. 콘텐츠 마케팅 (Content)
   → 무료 가치 제공 → 신뢰 → 전환
   
3. 유료 광고 (Paid Ads)
   → 빠른 스케일 가능, 검증된 후 투입
   
4. 리퍼럴 (Referrals)
   → 기존 고객이 새 고객 데려옴
   → 가장 낮은 CAC, 가장 높은 LTV
```

### 콘텐츠 마케팅 Hormozi 법칙
```
무료 콘텐츠 전략:
- 최고의 정보를 무료로 뿌려라
- "이걸 무료로 줘도 돼?" 싶을 때 줘라
- 무료 정보의 질이 유료 전환의 설득력

"Give away the secrets, sell the implementation"
→ 방법은 무료로, 실행(서비스/도구)은 유료로
```

### Hormozi 가격 책정 원칙
```
잘못된 생각: "싸게 팔면 더 많이 팔린다"
실제: 가격이 낮으면 인지 가치도 낮아짐

해결: 가격을 높이고 → 가치를 더 높게 증명하라
  - 비교 앵커링 사용 ("컨설팅 비용 100만원 vs 우리 30만원")
  - ROI 계산 제시 ("3개월 안에 원금 회수")
```

### 서비스 출력물 적용
```
오퍼 구성 공식:
[핵심 결과물] + [시간 단축] + [노력 제거] + [보증]

예: "AI가 30분 만에 1달치 콘텐츠 계획 완성
    + 6개 플랫폼 동시 출력
    + 수정 횟수 무제한
    + 만족 못하면 전액 환불"
```

## 3. Al Ries & Jack Trout — The 22 Immutable Laws of Marketing (1993)

### 핵심 철학
> "마케팅은 제품의 싸움이 아니다. 인식의 싸움이다."

### 22가지 법칙 (핵심 10개)

| 법칙 | 이름 | 내용 | 서비스 적용 |
|------|------|------|------------|
| 1 | **리더십 법칙** | 1등이 되는 것이 더 좋은 것이 되는 것보다 낫다 | 틈새 카테고리에서 1등 포지션 확보 |
| 2 | **카테고리 법칙** | 1등이 될 수 없다면 새 카테고리를 만들어라 | "AI 유튜브 코치" 새 카테고리 |
| 3 | **마음 법칙** | 시장보다 마음속에 먼저 들어가라 | 첫 인상/첫 콘텐츠가 포지션 결정 |
| 4 | **인식 법칙** | 마케팅은 제품이 아닌 인식의 싸움 | "더 좋다"가 아니라 "다르다" 강조 |
| 5 | **집중 법칙** | 마케팅에서 가장 강력한 것은 한 단어를 소유하는 것 | 브랜드 = 단어 하나 |
| 6 | **독점 법칙** | 두 기업이 같은 단어를 소유할 수 없다 | 경쟁사가 쓰는 단어 피하기 |
| 9 | **반대 법칙** | 1등을 목표로 한다면, 1등의 반대 전략을 써라 | 1등이 크면 나는 작게/친근하게 |
| 13 | **희생 법칙** | 무언가를 얻으려면 무언가를 포기해야 한다 | 타겟 좁히기 = 임팩트 강화 |
| 17 | **예측 불가 법칙** | 경쟁사의 계획을 예측할 수 없다 | 유연한 전략 유지 |
| 22 | **자원 법칙** | 자원 없이는 아이디어도 소용없다 | 실행 자원 확보가 전략보다 중요 |

### 카테고리 설계 공식
```
포지셔닝이 막힌 경우:
1. 기존 카테고리에서 경쟁 → 어려움
2. 새 카테고리 정의 → 자동으로 1등

예시:
- 스마트폰 아님 → "터치 컴퓨터" (아이폰)
- 에너지음료 아님 → "날개를 달아줘" (레드불)
- AI 콘텐츠 도구 아님 → "40주 강의 대리실행 AI"
```

### 집중 법칙 실전
```
강력한 브랜드 = 단어 하나 소유

볼보 = 안전
BMW = 드라이빙
FedEx = 빠른 배송

AI 오토파일럿 서비스가 소유할 단어: "수익화 자동완성"
```

## 4. Jay Conrad Levinson — Guerrilla Marketing (1984)

### 핵심 철학
> "대기업의 마케팅 전략이 아니라 소기업이 창의성과 에너지로 이기는 방법"

### 게릴라 마케팅 7가지 무기

```
1. 마케팅 캘린더
   → 52주 콘텐츠 스케줄 사전 확정
   → 일관성이 인지도보다 중요

2. 아이덴티티 마케팅
   → 제품/서비스가 아닌 정체성 판매
   → "우리는 ~하는 사람들을 위한 브랜드"

3. 바이럴 씨앗 심기
   → 모든 콘텐츠에 공유 요소 내장
   → "이거 아는 사람한테 알려줘야지" 유발

4. 크로스 프로모션
   → 보완적 파트너와 협업
   → 비용 없이 노출 2배

5. 무기한 팔로업
   → 한 번 접촉한 리드를 끝까지 추적
   → 대부분의 판매는 7번째 접촉에서 발생

6. 무료 샘플/체험
   → Hopkins 이론과 동일 — 무료 체험이 최강 판매

7. 측정 가능한 마케팅
   → 모든 활동에 추적 코드
   → ROI 계산 가능한 것만 지속
```

### 게릴라 콘텐츠 전략
```
소규모 크리에이터/서비스의 생존 공식:

1. 일관성 > 완성도
   → 완벽한 1개 < 꾸준한 52개

2. 틈새 지배
   → 전체 시장 10%보다 특정 틈새 100%

3. 관계 자산
   → 구독자는 고객이 아니라 친구
   → 친구에게 팔면 거부감 없음

4. 속도
   → 대기업보다 빠른 실행이 무기
   → 트렌드 선점 → 알고리즘 수혜
```

### 게릴라 vs 전통 마케팅 비교

| 요소 | 전통 마케팅 | 게릴라 마케팅 |
|------|------------|--------------|
| 투자 | 돈 | 시간/에너지/창의성 |
| 측정 | 판매량 | 관계/LTV |
| 성장 | 직선적 | 기하급수적 (바이럴) |
| 초점 | 신규 고객 | 기존 고객 유지 |
| 무기 | 광고비 | 커뮤니티/콘텐츠 |

## 통합 서비스 적용 공식

### 오퍼 설계 마스터 프레임 (Hormozi × Kotler × 22 Laws)

```
Step 1 (Kotler STP): 타겟 명확화
  → "누구에게 팔 것인가" 정의

Step 2 (22 Laws 카테고리): 새 카테고리 선점
  → "우리만의 카테고리 이름" 정의

Step 3 (Hormozi 오퍼): 거부 불가능한 패키지 설계
  → 드림 아웃컴 + 빠름 + 쉬움 + 보증

Step 4 (Guerrilla 실행): 일관성 있는 배포
  → 52주 캘린더 + 바이럴 요소 내장
```

### 가격/가치 포지셔닝
```
비싸 보이지 않으면서 가치 있어 보이는 방법 (Hormozi):
1. 비교 앵커 제시 → "기존 방법은 ~원, 우리는 ~원"
2. ROI 계산 → "3개월 내 ~만원 회수"
3. 시간 환산 → "직접 하면 100시간, AI로 10분"
4. 보증 → "효과 없으면 전액 환불"
```
